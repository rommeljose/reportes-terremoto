# Reportes ciudadanos — Doblete de Yumare 2026 (API pública)

Copia de preservación humanitaria de los reportes de daño de **terremotovenezuela.com**
(su base de datos fue purgada). Datos de acceso abierto, CORS habilitado, servidos por GitHub Pages.

**Documentación:** https://rommeljose.github.io/reportes-terremoto/api.html

| Endpoint | Contenido |
|---|---|
| `reportes.geojson` | FeatureCollection (QGIS, Google My Maps, Leaflet) |
| `reportes.json` | arreglo de objetos |
| `reportes.csv` | tabla (Excel) |
| `meta.json` | conteos, fecha de corte, esquema |

Fotos servidas desde `https://rommeljose.github.io/Sismo_fotos/`. Atribución: terremotovenezuela.com.
